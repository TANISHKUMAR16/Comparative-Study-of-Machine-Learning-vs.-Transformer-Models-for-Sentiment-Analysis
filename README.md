# Comparative Study of Machine Learning vs Transformer Models for Sentiment Analysis

## Overview
This project compares traditional Machine Learning models with DistilBERT for binary sentiment classification using IMDB and Twitter (Sentiment140) datasets.

## Models
- Logistic Regression
- SVM
- Random Forest
- XGBoost
- DistilBERT

## Technologies
- Python
- Scikit-learn
- Hugging Face Transformers
- Pandas
- NLTK
- XGBoost

## Datasets
- IMDB Movie Reviews
- Twitter Sentiment140

## Results
The repository includes comparison graphs and evaluation metrics (Accuracy and F1 Score).
